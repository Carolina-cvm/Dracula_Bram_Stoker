<h1 align="center">🧛‍ Drácula de Bram Stoker (1897) 🧛‍</h1>
<h3 align="center">📋 Resumo do livro, personagens, adaptações do cinema e curiosidades.</h3>
<h4 align="justify">Desenvolvi essa página utilizando HTML e CSS, inspirada no filme Drácula (1992). Nesta página, apresento um resumo detalhado do livro, informações sobre os personagens principais, adaptações cinematográficas e algumas curiosidades intrigantes. O design e o conteúdo foram cuidadosamente elaborados para proporcionar aos visitantes uma imersão completa no universo envolvente de Drácula.</h4>
